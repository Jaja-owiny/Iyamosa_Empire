# Iyamossa Empire Website Portfolio

A comprehensive static website portfolio for Iyamossa Empire, featuring modern design, responsive layout, and professional presentation of services across digital marketing, web design, photography, automation, graphic design, furniture sales, and construction.

## 🚀 Features

### ✅ Completed Features
- **Responsive Design**: Mobile-first approach with breakpoints for all devices
- **Modern UI/UX**: Professional design with smooth animations and hover effects
- **Navigation**: Sticky header with smooth scrolling and mobile hamburger menu
- **Home Page**: Complete with hero section, services showcase, stats, and contact form
- **About Page**: Company story, timeline, mission/vision, values, and team preview
- **Interactive Elements**: Scroll animations, parallax effects, and micro-interactions
- **SEO Optimized**: Meta tags, structured data, and accessibility features
- **Performance**: Optimized images, lazy loading, and efficient CSS/JS

### 🔄 Placeholder Pages (Ready for Content)
- **Team Page**: Individual team member profiles with detailed bios
- **Projects Page**: Portfolio galleries categorized by service type
- **Contact Page**: Dedicated contact form with map integration

## 🛠 Technology Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with CSS Variables and Flexbox/Grid
- **Icons**: Font Awesome 6.4.0
- **Fonts**: Google Fonts (Poppins)
- **Animations**: CSS animations with Intersection Observer API

## 🚀 Getting Started

### Prerequisites
- Python 3.7+
- Flask

### Installation
1. Navigate to the project directory:
   ```bash
   cd iyamossa-empire
   ```

2. Install dependencies:
   ```bash
   pip install flask
   ```

3. Run the application:
   ```bash
   python app.py
   ```

4. Open your browser and visit:
   ```
   http://localhost:5000
   ```

## 🔧 Customization

### Adding New Pages
1. Create HTML template in `templates/`
2. Add route in `app.py`
3. Update navigation in `base.html`

### Modifying Styles
- **Colors**: Update `static/css/variables.css`
- **Components**: Modify `static/css/components.css`
- **Animations**: Adjust `static/css/animations.css`

### Adding Content
- **Images**: Place in `static/images/`
- **Team Members**: Update team section in templates
- **Projects**: Create project galleries
- **Services**: Modify service cards and descriptions
